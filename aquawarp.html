Aqua Warp

https://www.spigotmc.org/resources/aqua-warp.29982/

I. Introduction

This is my third plugin. What is this ? This plugin helps you create and manage warps very easy.


II. Feature

  • Create/Delete warps
  • Sign Warps
  • List warp
  • Lock/Unlock Warps
  • Check Warps
  • Edit Warps
  • Service charges
  • Warp GUI

III. Commands

  • /warp <name> [<player>]
  • /warps
  • /warps set <name> [<x> <y> <z> <yaw> <world>]
  • /warps del <name>
  • /warps list
  • /warps lock <name> <password>
  • /warps unlock <name>
  • /warps feetp <name> <cost (for unlocked warp)> <cost (for locked warp)>
  • /warps gui
  • /warpadmin reload
  • /aquawarp reload
